◆ debounce_pressRelease()

eDebounceStatus_t debounce_pressRelease ( sDebounceHelper_t *const  pMe)

Debounce algorithm that handles both press time and release time.

This function needs to be called periodically at the rate set in debounce_Init routine The result of if the button was pressed or not is updated in member r_IsKeyPressed of sDebounceHelper_t The result of it the key was changed or not is updated in member r_IsKeyChanged of sDebounceHelper_t

Parameters
pMe: handler of type pointer to sDebounceHelper_t
Returns
eDebounceStatus_t return status

‍During first pass count needs to be set to release count

‍read actual keys here

‍if the current key press state and global key press state are same, update count

‍setup the timer to monitor next change

‍Wait unit the key state is stabilized

‍Accept the change and update the result booleans accordingly

static uint16_t debounce_setDebounceIgnoreMask ( sDebounceHelper_t *const  pMe)
static

utility function to prepare debounce mask

The debounce release mask determines how much debouncing is added, This is to be a multiple of button polling interval
A debounce mask of 0xF000 (0b 1111 0000 0000 0000) would mean a debouncing period of 4 * (button read time)
