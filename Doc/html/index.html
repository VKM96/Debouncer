ButtonDebounce 1.0
A collection of Debouncing algorithms suitable for embedded environments
Debouncer 

Collection of Debouncing strategies from the internet suitable for embedded systems. The algorithms currently implemented are from this article by Jack Ganssle https://www.embedded.com/my-favorite-software-debouncers/

Code Structure

  1. Source files are present in Src
  2. Interface files are present in inc
  3. PLease refer doxygen generated HTML documentation in the Doc/html subfolder for implementation details
  4. The function entry point is Main.c

Usage

  1. Initialization of the debugger is achieved through debounce_Init in Debounce.h
  2. At initialization the function for reading raw button input needs to be passed to debounce_Init.
  3. debounce_pressRelease or debounce_bitTrain need to be called periodically using an interrupt or task, Please refer to the debounce_Init section for details
  4. debounce_isKeyPressed is used to read debounced buttons
